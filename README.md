### Installation
```bash
$ [sudo] pip install mac-login-items
```

```bash
$ [sudo] npm i -g mac-login-items
```

#### Examples
```bash
$ login-items names
Google Chrome
Windscribe
```

```bash
$ login-items paths
/Users/username/Applications/Google Chrome.app
/Applications/Windscribe.app
```

```bash
$ login-items add /Applications/CCMenu.app
```

```bash
$ login-items rm CCMenu
```
